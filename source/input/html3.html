        Supported Browsers 2017

        We suggest you use the latest versions of Chrome, Safari or Firefox to access duo.

          

        We do not recommend the use of Internet Explorer or Microsoft Edge if other browsers are available.

         

        Blackboard publish a list of supported browsers.

        First Visit?

        When you log in for the first time, you will see a dark grey overlay and a prompt to complete your user profile - part of the social tools built into duo.
        We strongly recommend that you click the I'll do it later link to dismiss this page.
        Then read this profile creation guide on the LTT Help blog for advice about profiles. Note that you don't need to create a profile to use duo.
